# Athena-Revision 🏛️

AI-powered study revision app for Athena.

## Concept

1. 📸 **Capture** - Take photos of textbook pages
2. 📝 **Extract** - AI extracts content to Markdown
3. 🎯 **Practice** - Generate quiz questions for revision

## How It Works

### For Parents/Teachers (Cui):
1. Send photos of textbook pages to Goblin
2. Goblin extracts and stores content as Markdown
3. Select quiz type and difficulty

### For Athena:
1. Open the app
2. Choose subject and chapter
3. Select quiz type (MCQ, True/False, Fill-in-blank, Short Answer)
4. Practice and learn!

## Quiz Types

- **MCQ** - Multiple Choice Questions
- **True/False** - True or False statements
- **Fill-in-Blank** - Complete the sentence
- **Short Answer** - Brief written responses
- **Essay** - Longer analytical questions

## Tech Stack

- Frontend: Vanilla HTML/CSS/JS
- Storage: GitHub Repository
- AI: Goblin (Claude-powered)
