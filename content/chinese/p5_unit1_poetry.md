# 小五中文科 第一課：古詩朗讀

## 📖 課文內容

### 詠柳
**唐 · 賀知章**

碧玉妝成一樹高，  
萬條垂下綠絲絛。  
不知細葉誰裁出，  
二月春風似剪刀。

---

## 📝 詞語解釋

| 詞語 | 意思 |
|------|------|
| 碧玉 | 青綠色的玉石，比喻柳葉的顏色 |
| 妝成 | 打扮成、裝飾成 |
| 絲絛 | 用絲編成的帶子，比喻下垂的柳枝 |
| 細葉 | 細長的葉子 |
| 裁出 | 剪裁出來 |
| 二月 | 春季的第二個月 |
| 春風 | 春天的風 |
| 似 | 像、好像 |

---

## 🎯 重點理解

### 1. 詩歌主題
- 描寫春天的楊柳
- 讚美大自然的奇妙

### 2. 修辭手法
- **比喻**：把柳葉比作「碧玉」
- **擬人**：春風「似剪刀」形象生動

### 3. 詩歌大意
- 第一、二句：描寫柳樹的形態
- 第三、四句：提出疑問，引出答案是春風

---

## 📚 延伸學習

### 近義詞
- 碧玉 → 翠綠、嫩綠
- 垂下 → 懸掛、下垂

### 反義詞
- 高 → 低
- 細 → 粗
